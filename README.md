In this project, we have implemented a time-varying convolutional encoder (K=7, rate 1/2) driven by chaos in such a way that its generator polynomials are dynamically varied based on the synchronized logistic map series, thereby giving us secure physical layer protection without compromising the forward error correction. The functions provided in the project enable the generation of chaotic control bits, encoding of binary information using either static NASA polynomials (133,171) or varying chaotic polynomials, and decoding of the received bits with the help of a 64-state hard-decision Viterbi decoder synchronized with the same chaos. In order to operate the proposed system, start by testing test_encoder function for the verification of static encoder, followed by test_chaos_gen for the verification of logistic map series and finally test_full_system to confirm that perfect decoding is achieved. To analyze the performance of the system, run ber_simulation_200errors, where simulation of BPSK over AWGN with more than 200 bit errors at each SNR level is done to provide smooth BER comparison between fixed and chaos-driven codes.
